## CSS 动画

- 🔉说明

  - 一些 CSS 属性是可以有动画效果的，
    - 这意味着它们可以用于<font color='#FF2D00'>动画</font>和<font color='#FF2D00'>过渡</font>。

  - 动画属性可以逐渐地
    - <u>从一个值变化到另一个值，</u>
      - 🍒例如
        - 尺寸大小、数量、百分比和颜色。
