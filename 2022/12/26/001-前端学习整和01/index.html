第一篇 网页为啥叫HTML？

  1. 网页是怎么形成的？

  2. 具体化：

    • 网页通过 浏览器 才能访问阅读
    • 网页是网站中的一页
    • 文件扩展名是.html

  3. 什么是HTML？

    • HTML 是用来描述网页的一种语言。
      • HTML指的是超文本标记语言（Hyper Text Markup Language)
      • HTML不是一种编程语言，而是一种标记语言（markup language）
      • 标记语言是一套标记标签（markup tag)

  4. 总结

    • 网页是前端人员开发人员写的，通过浏览器转换成普通人眼中的美丽页面
    • 网页的文件扩展名是.html
    • HTML使用标记标签来描述网页，包括视频、音频、图片…

  • 一般情况下，一个 HTML 标签由开始标签、属性、内容和结束标签组成

  • 标签的名称不区分大小写，但大多数属性的值需要区分大小写，如下所示：

虽然 HTML 标签在语法上不区分大小写，但是为了规范和专业，强烈建议在定义标签时一律采用小写。

  • 当使用浏览器打开我们编写的 HTML 文档时，浏览器会从上到下依次读取文档中的内容，并根据 HTML 标签和属性将标签中的内容呈现在浏览器中。

2.一个 HTML 文档中必须具有一些基本的标签，以便浏览器区分普通文本和 HTML 文档。您可以根据想要实现的效果使用任意数量的标签，但有以下几点需要注意：

  • 所有 HTML 标签都必须放在尖括号< >内；

  • HTML 中不同的标签可以实现不同的效果；

  • 如果使用了某个标签，则必须使用对应的结束标签来结尾（自闭合标签除外）。

第七篇 HTML<div>和<span>标签

  1. <div> 标签

    • HTML <div>元素是块级元素，它可用于组合其他 HTML 元素的容器。

    • <div>元素没有特定的含义。除此之外，由于它属于块级元素，浏览器会在其前后显示折行。

    • 如果与 CSS 一同使用，<div>元素可用于对大的内容块设置样式属性。

  2. <span>标签

    • HTML<span>元素是内联元素，可用作文本的容器
    • <span>元素也没有特定的含义。
    • 当与 CSS 一同使用时，<span>元素可用于为部分文本设置样式属性。

  3. HTML 分组标签

标签 描述
<div> 定义了文档的区域，块级 (block-level)
<span> 用来组合文档中的行内元素， 内联元素(inline)
