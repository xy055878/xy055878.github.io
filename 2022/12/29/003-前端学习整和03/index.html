第一篇 超链接标签

↓ 属性代码如下：

  1. 标签定义及使用说明

    • <a> 标签定义超链接，用于从一个页面链接到另一个页面。

    • <a> 元素最重要的属性是 href 属性，它指定链接的目标。

  2. 在所有浏览器中，链接的默认外观如下：

    • 未被访问的链接带有下划线而且是蓝色的

    • 已被访问的链接带有下划线而且是紫色的

    • 活动链接带有下划线而且是红色的

属性 属性值 描述
target 规定在何处打开目标 URL。仅在 href 属性存在时使用。
_blank 新窗口打开
_parent 在父窗口中打开链接（页面中使用框架才有用）
_self 默认，当前页面跳转
_top 以整个浏览器作为窗口显示新页面（突破了页面框架的限制）
framename 自定义框架名

注意：设置 target 属性时， top 与 parent 的打开方式十分类似，需仔细区分。

比如网 A 中镶嵌了 iframe 网页 B，网页 B 又镶嵌了 iframe 网页C。

  • 如果网页 C 中连接设置 target=_parent，则跳转将网页 B 去掉直接在 A 中嵌入网页 C 中链接页面。
  • 如果网页 C 中 target=_top ，则直接跳出所有 iframe 框架，直接转向 C 中链接页面。

第二篇 table表格

  • 表格由 <table> 标签来定义。

    • 每个表格均有若干行（由 <tr> 标签定义），每行被分割为若干单元格（由 <td> 标签定义）。

    • 字母 td 指表格数据（table data），即数据单元格的内容。

    • 数据单元格可以包含文本、图片、列表、段落、表单、水平线、表格等等。

3.table表格的相关属性

属性 描述 属性值
width 宽度 数值[px] 或 %(相对于父元素的宽度) 500[px] 或 50%
height 高度 数值[px] 或 %(相对于父元素的高度) 500[px] 或 50%
border 边框 数值[px] D：0
bordercolor 边框颜色
bgcolor 背景颜色
align 水平对齐 D：left或right或center
cellspacing 单元格与单元格之间的间距 D：2
cellpadding 单元格与内容之间的空隙 D：1

4.HTML 表格和边框属性

  • 如果不定义边框属性，表格将不显示边框。

  • 有时这很有用，但是大多数时候，我们希望显示边框。

5.行tr的属性

属性 描述 属性值
height 高度 数值[px] 或 %(相对于父元素的高度) 500[px] 或 50%
bgcolor 背景颜色
align 文字水平对齐 D：left或right或center
valign 文字垂直对齐 top或D：middle或bottom

6.单元格td属性

属性 描述 属性值
width 宽度 数值[px] 或 %(相对于父元素的宽度) 500[px] 或 50%
height 高度 数值[px] 或 %(相对于父元素的高度) 500[px] 或 50%
bgcolor 背景颜色
align 文字水平对齐 D：left或right或center
valign 文字垂直对齐 top或D：middle或bottom
rowspan 跨行 n（数值） 下一行少写n-1个 td 或 th 否则会多出一列
colspan 跨列 n（数值） 本行少写n-1个 td 或 th 否则会多出一列
