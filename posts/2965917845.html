CSS3 过渡

CSS3 可以创建过渡，它可以取代许多网页动画图像、Flash 动画和 JavaScript 实现的效果。

  • 🔥案例

    • 用鼠标移过下面的元素。

  • 🔉说明

    • CSS3 过渡(transition)是通过定义元素逐渐从起点变化到结束点的状态 ，
      • 在一定的时间内，实现元素平滑地过渡的一种补间动画机制。
      • 你可以让属性的改变过程持续一段时间，而不是立即生效。
        • 要做到这一点，你必须指定两项内容 ➾
          1. 指定要添加效果的CSS属性。
          2. 指定效果的持续时间。
      • 🔔提示
        • 所谓补间动画又叫作中间帧动画、渐变动画,
          • 只要建立起始点和结束点的画面，
          • 中间部分由软件自动生成，省去了中间动画制作的复杂过程。

CSS3 transition-property属性

用于规定应用过渡效果的CSS属性的名称。

  • ❗注意

    • 请始终指定transition-duration属性，
      • 📌原因
        • 持续时间默认值 ➾ 0，🪶即transition不会有任何效果。

  • 🎯特性

    默认值all
    继承no
    JavaScript 语法object.style.transitionProperty=“width,height”

  • 语法

    • 🍬解析
      1. none ⥤ 没有属性会获得过渡效果。
      2. all ⥤ 所有属性都将获得过渡效果。
      3. property ⥤ 定义应用过渡效果的 CS属性名称列表，列表需以{逗号分隔}。

  • 🔥案例

CSS3 transition-duration 属性

用于规定完成过渡效果需要花费的时间（以秒〔s〕或毫秒〔ms〕计）。

  • 🎯特性

    默认值0
    继承no
    JavaScript 语法object.style.transitionDuration=“5s”

  • 语法

    • 🍬解析
      • time ⥤ 过渡时间（以秒〔s〕或毫秒〔ms〕计）。
        • 默认值 ➾ 0，🪶即transition不会有任何效果。

CSS3 transition-timing-function 属性

用于规定过渡效果的速度曲线。

  • 🎯特性

    默认值ease
    继承no
    JavaScript 语法object.style.transitionTimingFunction=“linear”

  • 语法

    • 🍬解析

      1. linear

        • 匀速的过渡效果。

      2. ease

        • 慢速开始，而后快速，慢速结束的过渡效果。

      3. ease-in

        • 慢速开始的过渡效果。

      4. ease-out

        • 慢速结束的过渡效果。

      5. ease-in-out

        • 慢速开始，慢速结束的过渡效果。

      6. cubic-bezier(n,n,n,n)

        • 在一个贝塞尔函数中，自定义速度曲线值。
        • 来回的缓冲效果{cubic-bezier(0.68, -0.55, 0.27, 1.55)}。

CSS3 transition-delay 属性

用于规定过渡效果何时开始（以秒〔s〕或毫秒〔ms〕计）。

  • 🎯特性

    默认值0
    继承no
    JavaScript 语法object.style.transitionDelay=“2s”

  • 语法

    • 🍬解析

      • time ⥤ 延迟时间或截断时间。

        • ⏳详述 ➾ 非0的时候必须设置单位 ➾ 秒〔s〕或毫秒〔ms〕。
          1. 默认值 ➾ 0 ⥤ 不延迟，立即显示过渡效果。
          2. 正数 ⥤ 按照设置的时间延迟显示过渡效果。
          3. 负数 ⥤ 从设置时间开始显示过渡效果，设置时间之前的动作被截断。

      • 🔥案例

        • transition: all 2s 1s; {transition-delay: 1s;}

        • transition: all 2s -1s; {transition-delay: -1s;}

CSS3 transition 属性

以下四个属性的简写属性 ➾ 

  1. transition-property
  2. transition-duration
  3. transition-timing-function
  4. transition-delay

🎯特性

默认值all 0 ease 0
继承no
JavaScript 语法object.style.transition=“width 2s”
