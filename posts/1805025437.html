物理机的网络知识

一、IP地址

类似于手机号。

  • 英译: \text{: }: IPADDR。
  • ✨作用
    • 使得设备(譬如: \text{: }: 电脑、服务器)之间相互通信。
  • 组成
    • 4段•点分十进制 ➾ xxx.xxx.xxx.xxx
      • 🔔提示
        1. IP地址可以随意更改。
        2. 点分: \text{: }: 每一段以点来划分。
        3. 每段范围: \text{: }: 0~255，譬如: \text{: }: 0.0.0.0~255.255.255.255

⑴查看本机IP地址

1.方法一{适用于Win11}

  1. 进入设置。

  2. 点击网络和Internet。

    1. 🔦方法一

      1. 点击高级网络设置。

      2. 点击更多网络适配器选项。

      3. 双击WLAN。『💫示例: \text{: }: 无线网』

        • 🔔提示

          1. 如果是通过有线上网，那么请双击『以太网』

          2. 如下为虚拟网络，只有装了虚拟机才会出现。

      4. 点击详细信息，查看IP地址等。

    2. 🔦方法二

      1. 点击属性。

      2. 查看IP地址。

2.方法二

  1. 按Win + R打开“运行”窗口，输入control打开控制面板。

  2. 点击网络和Internet。

  3. 点击查看网络状态和任务。

  4. 点击WLAN/以太网(有线)。『💫示例: \text{: }: 无线网』

  5. 如下同上。

3.方法三

  1. 按Win + R打开“运行”窗口，输入cmd打开“命令提示符”窗口。

  2. 在命令行上运行ipconfig命令『💫示例: \text{: }: 有线网』

    • 🔔提示
      1. 物理网卡➾ 以太网适配器 以太网 2:
      2. 虚拟网卡➾ 以太网适配器 VMware Network Adapter VMnet1:
      3. 虚拟网卡➾ 以太网适配器 VMware Network Adapter VMnet8:

  3. 打开多个微信。在一台设备上，开启多个微信登录界面。

    • 📌原因
      • 中文路径。
    • 🔔提示
      1. 同一个账户不能登录多个微信。
        • 📌原因
          • ﹝微信官方规定﹞在一个新设备上登录一个账户，旧设备将登出此账户。
      2. 但一个在同一台设备上，可以登录多个人的账户。
1º查看本机详细网络信息

  1. 按Win + R打开“运行”窗口，输入cmd打开“命令提示符”窗口。

  2. 在命令行上运行ipconfig -all命令

⑵ping命令

在Windows下的“命令提示符”窗口的命令行上，运行ping命令。

  • ✨作用

    • 检查网络是否通畅。

  • 🧮格式

  • 🔥案例

    1. 非正常通信。

      • 📌原因
        1. 我的IP地址: \text{: }: 192.168.43.226
          • ➾ 两个IP地址不在同一个网段(前三段不相同)。
        2. 他的IP地址: \text{: }: 192.168.17.1
      • 📌原因
        • 在这个网段(192.168.43.225)没有设备。

    2. 正常通信。

      • 👁️‍🗨️分析
        • 我可以上外网。
        • 📌原因
          • 百度的IP地址是公网IP地址，任何设备都可以访问。

⑶IP地址的分类

1.公网IP地址

任何设备都可以访问。类似于公共厕所，所有人都可以使用。

  • 🍒例如
    • 访问百度。

2.私网IP地址

只有在同一「局域网」内的设备才可以访问。类似于自家厕所，自家人才可以使用。

3.设备上网流程

⑷IP地址的分配方式

1.静态IP

手动配置IP地址。『IP地址将 ➾ 固定』

  • 🔦方法

    1. 进入设置。

    2. 点击网络和Internet。

      1. 🔦方法一

        1. 点击高级网络设置。

        2. 点击更多网络适配器选项。

        3. 双击WLAN。『💫示例: \text{: }: 无线网』

          • 🔔提示

            1. 如果是通过有线上网，那么请双击『以太网』

            2. 如下为虚拟网络，只有装了虚拟机才会出现。

        4. 点击属性。

        5. 双击IPv4协议。

        6. 手动设置IP地址。

          1. 原IP地址。

          2. 现IP地址。

        7. ping 192.168.43.250

      2. 🔦方法二

        1. 点击属性。

        2. 查看IP地址。

        3. 手动设置IP地址。

二、子网掩码

类似于运营商，决定了手机号能给谁打电话。

  • 英译: \text{: }: NETMASK。
  • ✨作用

    • 划分网段，判断IP地址是否在同一个「局域网」内。

    • 👁️‍🗨️分析

      1. 255.255.255.0

        1. 255.255.255➾ 网络位 ➾ 又称网段。
          • 🔔提示
            • 如果两个IP地址的网段相同，那么它们就在同一个「局域网」内 ➾ 两台设备之间可以通信。
        2. 0➾ 主机位。
        3. IP地址的「前三位」必须相同才能通信。
          • 🍒例如
            1. 可以通信
              1. IP地址$\text{: }$10.0.0.7
              2. 网关$\text{: }$10.0.0.8
            2. 不可以通信
              1. IP地址$\text{: }$10.0.0.7
              2. 网关$\text{: }$10.0.1.8

      2. 255.255.255.0

        1. 255.255➾ 网络位 ➾ 又称网段。
          • 🔔提示
            • 如果两个IP地址的网段相同，那么它们就在同一个「局域网」内 ➾ 两台设备之间可以通信。
        2. 255.0➾ 主机位。
        3. IP地址的「前两位」必须相同才能通信。
          • 🍒例如
            1. 可以通信
              1. IP地址$\text{: }$10.3.5.7
              2. 网关$\text{: }$10.3.6.8
            2. 不可以通信
              1. IP地址$\text{: }$10.3.5.7
              2. 网关$\text{: }$10.4.6.8

三、网关

类似于出国时，要经过的海关。

  • 英译: \text{: }: GATEWAY。

  • ✨作用

    • 让设备可以访问局域网以外的网站。

  • 🔔提示

    1. 网关其实就是路由器的IP地址。

    2. 将网关删除(不配网关)，在同一个局域网内的设备是可以通信的，但是访问不了外网。

      • 🍒例如

        • 一台设备没有配置网关，它将无法访问百度。

          1. 无网关

          2. 有网关

1.更改路由器的IP地址

当路由器的IP地址更改后，本设备的网关必须和路由器的IP地址相同。不然将无法与外网通信。

四、DNS

全称: \text{: }: domain name server(域名解析服务)。

  • ✨作用
    • 将域名解析为一个IP地址。
    • 🔔提示
      • 其实浏览器是不认识www.baidu.com的，但是DNS服务就可以将 ➾ 
        • www.baidu.com解析为180.101.50.188
  • 域名解析
    1. 正向解析
    2. 反向解析

五、DHCP

自动获取IP地址的服务。

  • 一般是在路由器上，如果开启了DHCP，那么所有连接路由器的设备，都可以自动获取到。
  • ❗注意
    • 它也能自动获取子网掩码、网关、DNS。
  • 💫示例
    • 阿里云的DHCP服务器
      1. 223.5.5.5
      2. 223.6.6.6

六、物理地址

类似于每个人的身份证，用于在网络中，唯一标示一个网卡。

  • 英译: \text{: }: MAC地址。

  • 🔔提示

    • 一台设备若有一或多个网卡，则每个网卡都需要并会有一个唯一的MAC地址。
