＊、修仙小故事

有一天张三在做运维，加班途中突然穿越到了神话大陆，由此踏上了修仙之路。

  1. 张三四处走走、逛逛: \text{: }: cd(进入某个目录)。
  2. 张三查看四周的环境: \text{: }: ls(列表显示目录内容)。
  3. 张三突然发现手中有手机，打开手机看了看北斗定位，查看自己的位 置: \text{: }: pwd(显示自己的所在目录)。
  4. 张三野外求生，自己搭建房子: \text{: }: mkdir(创建目录)，查看房子: \text{: }: ls(查看目录)。
  5. 张三置办家居，弄个床: \text{: }: touch(创建文件)，随着创建，发现修为提升到了筑基初期。
  6. 张三外出的时候，发现外面有好东西，ls(查看目录)看了下，决定搬回去: \text{: }: mv(移动)。
  7. 张三外出的时候，发现非常壮丽的房子，于是决定自己也建造一个差不多的房子: \text{: }: cp(拷贝)。
  8. 张三一天找到了一个峡谷大喊，回家后我要开始书写日记: \text{: }: vi(编辑与修改文件)。
  9. 张三修为提升的不错了，就放了个招式，结果家没了: \text{: }: rm(删除)，突然时空发生了扭曲回到了公司。

一、命令详解

⑴cd命令

进入到某个目录。

  • 语法

  • 🔥案例

    • 进入/tmp/目录。

      • 🍉结果

1.cd命令其他功能

  1. cd ~

    • 回到家目录。

  2. cd

    • 回到家目录。

  3. cd -

    • 进入上次进入目录。

    • 🔥案例

  4. cd .

    • 进入当前目录。
    • 🔔提示
      • 未来一般与复制命令、移动命令一起用。

  5. cd ..

    • 进入上级目录。

  6. cd ../..

    • 进入上上级目录。

⑵pwd命令

以「绝对路径方式」显示当前所在位置。

  • 语法

  • ✨作用

    • 当﹝不确定﹞当前所在位置时，可以通过pwd命令来查看当前目录的绝对路径。
    • 🔔温馨提示
      • 操作命令和目录时可以使用Tab键，自动补全命令、路径。

  • 🔥案例

    1. 进入/tmp/目录然后查看所在目录。

    2. 进入/etc/sysconfig/network-scripts/目录然后查看所在目录。

⑶ls命令

查看目录下的内容。

  • 🔔提示

    1. ls命令的输出信息可以进行「彩色」加亮显示，以区分不同类型的文件。

      • 🔥案例image-20240305121708034.png

        • 查看/根目录下的「简略」内容。

    2. 默认查看当前目录下的「简略」内容。

      • 🔥案例

        • 以默认方式查看/tmp目录下的「简略」内容。

    3. ls命令可以在当前目录查看其他目录中的内容。

      • 🔥案例

        • 在/etc目录中，查看/根和/tmp目录下的「简略」内容。

          • 📌原因
            • etc目录下没有tmp目录。所以推荐使用「绝对路径」。

  • 语法

    • ⚗️参数

      • 目录名

    • 🥏选项

      1. -l 「long」🌟

        • 以﹝长格式﹞显示详细内容，🍒例如文件时间、大小、所有者…

          • 🔥案例

            • 查看/根目录下的「详细」内容。

      • 🪞简写

        • 🔥案例

      1. -a「all」🌟

        • 显示所有内容，包含「隐藏」的内容。

        • 🔔提示

          • Linux下面所有以.开头的文件就是隐藏文件，默认不显示。

          • 🔥案例

      2. -h🌟

        • 以人类可读形式(单位转化)显示大小，默认按B(字节)显示。

        • 🔥案例

          • 查看/etc目录下文件的大小并且以人类可读形式显示。

            • 计算机大小单位
              1. 1B=8b(位)
              2. 1KB=1024B(字节)
              3. 1MB=1024KB
              4. 1GB=1024NB
              5. 1TB=1024GB
              6. 1PB=1024TB

      3. -t 「time」

        • 按照修改时间「正序排序」显示。

        • 🔥案例

          1. 默认按字母顺序排序。

          2. 按文件的修改时间排序。

      4. -r 「reverse」

        • 逆序排序，一般与-t一起用 ➾ ll -rt

        • 🔥案例

          • 按修改时间逆序排序。
