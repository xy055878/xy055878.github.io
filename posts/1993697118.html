Linux核心命令

＊、修仙小故事

有一天张三在做运维，加班途中突然穿越到了神话大陆，由此踏上了修仙之路。

  1. 张三四处走走、逛逛: \text{: }: cd(进入某个目录)。
  2. 张三查看四周的环境: \text{: }: ls(列表显示目录内容)。
  3. 张三突然发现手中有手机，打开手机看了看北斗定位，查看自己的位 置: \text{: }: pwd(显示自己的所在目录)。
  4. 张三野外求生，自己搭建房子: \text{: }: mkdir(创建目录)，查看房子: \text{: }: ls(查看目录)。
  5. 张三置办家居，弄个床: \text{: }: touch(创建文件)，随着创建，发现修为提升到了筑基初期。
  6. 张三外出的时候，发现外面有好东西，ls(查看目录)看了下，决定搬回去: \text{: }: mv(移动)。
  7. 张三外出的时候，发现非常壮丽的房子，于是决定自己也建造一个差不多的房子: \text{: }: cp(拷贝)。
  8. 张三一天找到了一个峡谷大喊，回家后我要开始书写日记: \text{: }: vi(编辑与修改文件)。
  9. 张三修为提升的不错了，就放了个招式，结果家没了: \text{: }: rm(删除)，突然时空发生了扭曲回到了公司。

一、命令详解

⑴cd命令

用于进入到某个目录。

  • 语法

    1. 🔥案例

      • 进入/tmp/目录。

1.cd命令其他功能

  1. cd ~

    • 回到家目录。

  2. cd

    • 回到家目录。

  3. cd -

    • 进入上次进入目录。

    • 🔥案例

  4. cd .

    • 进入当前目录。
    • 🔔提示
      • 未来一般与复制命令、移动命令一起用。

  5. cd ..

    • 进入上级目录。

  6. cd ../..

    • 进入上上级目录。

⑵pwd命令

用于以「绝对路径方式」显示当前所在位置。

  • 语法

  • ✨作用

    • 当﹝不确定﹞当前所在位置时，可以通过pwd命令来查看当前目录的绝对路径。

    • 🔥案例

      1. 进入/tmp/目录然后查看所在目录。

      2. 进入/etc/sysconfig/network-scripts/目录然后查看所在目录。

        • 🔔温馨提示
          • 操作命令和目录时，可以使用Tab键，自动补全「已存在的」命令、路径。

⑶ls命令

用于查看目录下的内容。

  • 语法

    1. ⚗️参数

      • 目录名，指定要显示列表的目录，也可以是具体的文件。

    2. 🔔提示

      1. ls命令的输出信息可以进行「彩色」加亮显示，以区分不同类型的文件。

        • 🔥案例

          • 查看/根目录下的「简略」内容。

      2. 默认查看当前目录下的「简略」内容。

        • 🔥案例

          • 以默认方式查看/tmp目录下的「简略」内容。

      3. ls命令可以在当前目录查看其他目录中的内容。

        • 🔥案例

          • 在/etc目录中，查看/根和/tmp目录下的「简略」内容。

            • 📌原因
              • etc目录下没有tmp目录。所以推荐使用「绝对路径」。

    3. 🥏选项

      1. -l 「long」🌟

        • 以﹝长格式﹞显示详细内容，🍒例如文件时间、大小、所有者…

          • 🔥案例

            • 查看/根目录下的「详细」内容。

        • 🪞简写

          • 🔥案例

    4. -a「all」🌟

      • 显示所有内容，包含「隐藏」的内容。

      • 🔔提示

        • Linux下面所有以.开头的文件就是隐藏文件，默认不显示。

        • 🔥案例

    5. -h🌟

      • 以人类可读形式(单位转化)显示大小，默认按B(字节)显示。

      • 🔥案例

        • 查看/etc目录下文件的大小并且以人类可读形式显示。

          • 计算机大小单位
            1. 1B=8b(位)
            2. 1KB=1024B(字节)
            3. 1MB=1024KB
            4. 1GB=1024MB
            5. 1TB=1024GB
            6. 1PB=1024TB

    6. -t 「time」

      • 按照「修改时间•正序排序」显示。

      • 🔥案例

        1. 默认按字母顺序排序。

        2. 按文件的修改时间排序。

    7. -r 「reverse」

      • 逆序排序，一般与-t一起用 ➾ ll -rt

      • 🔥案例

        • 按「修改时间•逆序排序」显示。

⑷mkdir命令

用于创建目录。

  • 语法

    1. ⚗️参数

      • 目录名: \text{: }: 指定要创建的目录列表，多个目录之间用空格隔开。

      • 🔥案例

        • 在当前路径下一次创建三个「同级」目录。

    2. 🔔提示

      1. 如果在目录名的前面「没有加」任何路径名，则在「当前目录下」创建由 ➾ 参数指定的目录。

        • 🔥案例

          • 在当前路径下新建一个目录。

      2. 如果给出了一个已经存在的路径，将会在该目录下创建一个指定的目录。

        • ❗注意

          • 在创建目录时，应保证新建的目录与它所在目录下的文件没有重名。

        • 🔥案例

    3. 🥏选项

      1. -p🌟

        • 若所要建立目录的上层目录目前 ➾ 尚未建立，则会一并建立上层目录。

          • 🔔提示

            • 不加选项的mkdir命令一次「只能」创建一层目录。

            • 🔥案例

        • 🔥案例

          • 新建嵌套目录。

      2. -v

        • 显示创建目录的过程。

        • 🔥案例

⑸touch命令

用于创建新的空文件。

  • 语法

    1. ⚗️参数

      • 文件名: \text{: }: 指定要创建的文件列表，多个文件之间用空格隔开。

    2. 🔔提示

      • 如果在文件名的前面「没有加」任何路径名，则在「当前目录下」创建由 ➾ 参数指定的文件。

        • 🔥案例

          • 在当前路径下新建一个文件。

    3. 🔥案例

      1. 在/home/tany/下面创建001.txt文件。

      • 🔔温馨提示
        • 在创建文件前，建议通过ls命令检查目录是否存在。

      1. 在/home/tany/目录下面创建002.txt和003.txt文件、004.txt和005.txt文件。

        1. 绝对路径方法

        2. 相对路径方法

    4. 🥏选项

⑹mv命令

用于将文件或目录从一个目录移动到另一个目录中，或对文件或目录重新命名。

  • 语法

    1. ⚗️参数

      1. 参数1: \text{: }: 源文件名/目录名。
      2. 参数2: \text{: }: 目标文件名/目录名。

    2. 🔥案例

      1. 目标移动

        1. 把/home/tany/001.txt移动到/tmp下。

        2. 把/home下的tany目录移动到/tmp下。

      2. 目标改名

        1. 将/home/tany目录下的001.txt文件改名为001.4k-fbi.avi文件。

        2. 把/home下的tany目录改名为tany001。

    3. 🥏选项

⑺cp命令

用于将源文件或目录复制到目标文件或目录中。

  • 语法

    1. ⚗️参数

      1. 参数1: \text{: }: 源文件名/目录名。
      2. 参数2: \text{: }: 目标文件名/目录名。

    2. 🔥案例

      • 将/home/tany目录下的001.txt文件复制到/home/tanb目录。

    3. 🥏选项

      1. -r

        • 递归复制，将指定目录下的所有文件与子目录一并处理 ➾ 复制目录。

        • 🔔提示

          • 默认情况下，cp命令不能复制目录。

          • 🔥案例

            • 将/home/tany目录复制到/tmp目录下 ➾ 『失败』。

        • 🔥案例

          1. 将/home/tany目录复制到/tmp目录下 ➾ 『成功』。

          2. 将/tmp/tany目录复制到/home目录(当前目录)下。

      2. -p

        • 保留源文件或目录的属性。

          • 🍒例如
            • 修改的时间不变。
            • 🔔提示
              • 直接使用cp命令会导致备份文件修改的时间改变为 ➾ 命令的执行时间。

        • 🔥案例

          1. cp

          2. cp -p

      3. -d

        • 复制时保留﹝软链接﹞。这里所说的软链接相当于Windows系统中的快捷方式。
        • 🔔提示
          1. 如果在复制软链接文件时，不使用-d选项，则 cp命令复制的是「源文件」。
          2. 如果在复制软链接文件时，使用-d选项，则 cp命令复制的是「软链接文件」。
            • ❗注意
              • -d选项对﹝硬链接﹞是无效的。

      4. -a

        • 相当于-rpd结合。

⑻rm命令

用于删除给定的文件和目录。

  • ❗注意

    • 使用rm命令要格外小心。
      • 📌原因
        • 一旦删除了一个文件，就无法再恢复它。
      • 所以，在删除文件之前，最好再看一下文件的内容，确定是否真要删除。

  • 语法

    1. ⚗️参数

      • 文件名/目录名。

    2. 🔥案例

      1. 删除/home/tany目录 ➾ 『失败』

      2. 删除/home/tany/001.txt文件

        • 🔔提示
          1. 取消删除方法
            1. Ctrl + C
            2. 字母n ➾ no。
          2. 确认删除方法
            • 字母y ➾ yes。

    3. 🥏选项

      1. -f

        • 强制删除文件或目录，不进行提示。

        • 🔥案例

      2. -r

        • 递归删除，将指定目录下的所有文件与子目录一并处理 ➾ 删除目录。

        • 🔥案例

⑼vi命令

大部分Linux发行版本内置的功能强大的纯文本编辑器。

  • 语法

    1. ⚗️参数

      • 文件名

      • 🔥案例

    2. 🥏选项

  • vi编辑器的工作模式

    1. 命令模式

      • 按Esc键进入，默认处于命令模式。

    2. 编辑模式

      • 按i/a/o键进入 ➾ -- INSERT --

    3. 末行模式

      • 在命令模式下，按:键进入。

        操作对应功能
        :q退出不保存文件，❗注意: \text{: }: 输入空格删除将无法成功。
        :q!强制退出不保存文件，❗注意: \text{: }: 输入空格删除则可以成功。
        :w保存文件文件。
        :wq保存并退出文件。
        :set nu将每行设置行号。
        :set nonu隐藏行号。
