Linux必会技术点

一、Linux命令行

Linux命令行指进入到Linux系统后，可以输入命令的地方。

⑴Linux命令行的组成

  • 🍬解析
    1. 用户名
      • 当前用户的名字。
    2. 主机名
      • 未来每个主机都要有名字。
      • ❗注意
        • 主机名请务必修改，若不修改默认是localhost。
    3. 当前位置
      1. 相对的一个位置，默认不会显示完整路径。『相对路径』
      2. ~表示用户的家目录。
    4. 提示符
      • 表示当前用户的身份。
        1. #表示root用户。
        2. $表示普通用户 。

二、Linux命令

Linux命令指对Linux系统进行管理的命令。

⑴Linux命令的格式

  • 🍬解析
    1. 命令
      • 在Linux系统下，可以通过命令控制整个Linux系统。『一般是单词或单词缩写』
    2. 选项
      • 同一个命令的不同的功能，🪶即在执行命令时，可以加上的条件。
    3. 参数
      • 命令处理的目标。
    4. 🔔温馨提示
      1. 一般就把选项和参数统称为 ➾ 参数即可。
      2. 并不是所有的常用的命令都需要加上参数和选项。

三、Linux与Windows目录结构上的区别

⑴Linux目录结构

在Linux中目录一切从根目录开始，Linux目录类似于一颗倒挂的大树。

⑵Windows目录结构

Windows目录一切从C盘或D盘[或？盘]开始。

  • 粘贴上一次剪切(删除)的内容: \text{: }: Ctrl + Y

  1. 清屏: \text{: }: Ctrl + L
  2. 补全命令、路径名: \text{: }: Tab

  3. 终止当前行输入的命令: \text{: }: Ctrl + C

  1. 锁屏: \text{: }: Ctrl + S

  3. 让当前正在运行的命令，后台挂起: \text{: }: Ctrl + Z

6.查找历史命令类快捷键

  1. 上一个命令: \text{: }: ↑
  2. 下一个命令: \text{: }: ↓

  2. 粘贴: \text{: }: Shift + insert

⑶Linux关机重启命令

1.关机

1ºshutdown命令

在1分钟后关闭/Linux系统。

  • 🔥案例

    • 🍉结果

  • 语法

    • 选项

      1. -h

        • 在1分钟后关闭Linux系统。

        • 参数

          • [时间]
            1. 设置多久时间后执行shutdown -h命令。
            2. 默认值: \text{: }: 1，以「分钟」计。

        • 🥦补充

          1. shutdown -h now

            • 立刻关闭Linux系统。

          2. shutdown -h 10

            • 在10分钟后关闭Linux系统。

            • 🔥案例

              • 🍉结果

          3. shutdown -h 21:35

            • 在今天21:35后关闭Linux系统。

      2. -c

        • 取消关闭/重启Linux系统。

          • 🍒举例说明

            • 当执行“shutdown -h 10”指令时，取消关闭Linux系统。

              • 🍉结果

2.重启

1ºshutdown命令

在1分钟后关闭/Linux系统。

  • 语法

    • 选项

      1. -r

        1. 在1分钟后重启Linux系统。

        2. 参数

          • [时间]

            1. 设置多久时间后执行shutdown -r命令。
            2. 默认值: \text{: }: 1，以「分钟」计。

          • 🥦补充

            1. shutdown -r now

              • 立刻重启Linux系统。

            2. shutdown -r 10

              • 在10分钟后重启Linux系统。

              • 🔥案例

                • 🍉结果

            3. shutdown -r 21:35

              • 在今天21:35后重启Linux系统。

      2. -c

        • 取消关闭/重启Linux系统。

          • 🔥案例

            • 当执行“shutdown -r 10”指令时，取消重启Linux系统。

              • 🍉结果

2ºreboot命令

立刻重启Linux系统。
