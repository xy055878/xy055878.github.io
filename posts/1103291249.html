糖丸2023-07-102023-12-03

Css3动画

Css3 可以创建动画{animation}，它可以取代许多网页动画图像、Flash 动画和 JavaScript 实现的效果。

Css3动画是什么？

动画是使元素从一种样式逐渐变化为另一种样式的效果。

transition和animation

异同

  1. 相同点

    • 都是随着时间改变元素的属性值。

  2. 不同点

    1. transition需要触发一个事件，才会随时间改变其Css属性，
    2. animation不需要触发任何事件，就会随时间改变其Css属性。

区别

  1. transition完成后，会保留过渡后的状态，而默认情况下，animation会跳至初始状态，
  2. transition是平滑过渡，而animation更精细，具体到每一帧都可以控制。

Css3的动画属性

Css3 @keyframes 规则

要创建 Css3 动画，你需要了解 @keyframes规则。

  • @keyframes规则内指定一个Css样式将逐步从当前的样式更改为新的样式。

    • 其可以用来定义Css动画的一个周期的行为，创建简单的动画。

  • 语法

    • 🍬解析
      1. animation-name『必需的』
        • 定义animation{关键帧}的名称。
      2. keyframes-selector『必需的』
        • 动画持续时间的百分比。
        • 合法值
          1. 0%{动画开始} ➾ 100%{动画结束}，
          2. from{和0%相同}，
          3. to{和100%相同}。
      3. css-styles『必需的』
        • 一个或多个合法的Css样式属性。
    • 🔔提示
      • 至少需要指定如下两个Css3的animation属性绑定向一个选择器。
        1. 用于规定动画的名称{animation-name}，
        2. 用于规定动画的时长{animation-duration}。

  • 🔥案例

    • 把“mymove”绑定到div元素。

Css3 animation-name 属性

用于绑定关键帧名称{@keyframes规则名称}。

  • 🎯特性

    默认值none
    继承no
    JavaScript 语法object object.style.animationName=“mymove”

  • 语法

    • 🍬解析
      1. none
        • 指定有没有动画。
      2. keyframe-name
        • 指定要绑定到选择器的关键帧名称。

Css3 animation-duration 属性

用于规定完成动画效果需要花费的时间{以秒〔s〕或毫秒〔ms〕计}。

  • 🎯特性

    默认值0
    继承no
    JavaScript 语法object object.style.animationDuration=“3s”

  • 语法

    • 🍬解析
      • time
        • 指定动画播放完成所花费的时间。默认值 ➾ 0，意味着没有动画效果。

Css3 animation-timing-function 属性

用于规定动画将如何完成一个周期。

  • 🔔提示

    • 速度曲线定义了动画从一套Css样式变为另一套所用的时间，速度曲线用于使变化更为平滑。

  • 🎯特性

    默认值ease
    继承no
    JavaScript 语法object object.style.animationTimingFunction=“linear”

  • 语法

  • 🍬解析

    • value ➾ animation-timing-function可选值。

      1. linear

        • 匀速的过渡效果。

      2. ease

        • 慢速开始，而后快速，慢速结束的过渡效果。

      3. ease-in

        • 慢速开始的过渡效果。

      4. ease-out

        • 慢速结束的过渡效果。

      5. ease-in-out

        • 慢速开始，慢速结束的过渡效果。

      6. cubic-bezier(n,n,n,n)

        • 在一个贝塞尔函数中，自定义速度曲线值。
        • 来回的缓冲效果{cubic-bezier(0.68, -0.55, 0.27, 1.55)}。

Css3 animation-delay 属性

用于规定动画效果何时开始{以秒〔s〕或毫秒〔ms〕计}。

  • 🎯特性

    默认值0
    继承no
    JavaScript 语法object object.style.animationDelay=“2s”

  • 语法

    • 🍬解析

      • time ⥤ 延迟时间或截断时间。

        • ⏳详述 ➾ 非0的时候必须设置单位 ➾ 秒〔s〕或毫秒〔ms〕。
          1. 默认值 ➾ 0 ⥤ 不延迟，立即显示动画效果。
          2. 正数 ⥤ 按照设置的时间延迟显示动画效果。
          3. 负数 ⥤ 从设置时间开始显示动画效果，设置时间之前的动作被截断。

Css3 animation-iteration-count 属性

用于规定动画的播放次数。

  • 🎯特性

    默认值1
    继承no
    JavaScript 语法object object.style.animationIterationCount=3

  • 语法

    • 🍬解析
      • value ➾ 播放次数。
        • 取值
          1. n ➾ 默认值: \text{: }: 1。『正数』
            • 一个数字，定义应该播放多少次动画。
          2. infinite
            • 指定动画应该播放无限次{永远}。

Css3 animation-direction 属性

用于规定动画的播放方向。

  • ❗注意

    • 如果动画被设置为只播放一次，该属性将不起作用。

  • 🎯特性

    默认值normal
    继承否
    JavaScript 语法object.style.animationDirection=“reverse”

  • 语法

    • 🍬解析
      1. normal『from ➾ to』
        • 正向播放动画。
      2. reverse『to ➾ from』
        • 反向播放动画。
      3. alternate
        1. 动画在奇数次{1、3、5…}正向播放，
        2. 动画在偶数次{2、4、6…}反向播放。
      4. alternate-reverse
        1. 动画在奇数次{1、3、5…}反向播放，
        2. 动画在偶数次{2、4、6…}正向播放。
      5. initial
        • 设置该属性为它的默认值，请参阅 initial。
      6. inherit
        • 从父元素继承该属性，请参阅 inherit。

Css3 animation-fill-mode 属性

用于规定{当动画不播放时或当动画延迟时或当动画播放完成时}，要应用到元素的样式。

  • 默认情况下，动画在动画执行之前和之后不会应用任何样式到目标元素。

    • ⏳详述
      1. Css动画在{第一个}关键帧播放完前，不会影响元素，
      2. Css动画在{最后一个}关键帧播放完后，停止影响元素。

  • 🎯特性

    默认值none
    继承否
    JavaScript 语法object.style.animationFillMode=“forwards”

  • 语法

  • 动画生命周期

    • 动画分为{初始状态、等待期、动画执行期、完成期}四个阶段。

      1. 初始状态

        • 指的是没有触发动画效果时，元素原本应该有的状态。

      2. 等待期

        • 指的是animation-delay属性设置的延迟期间。

          • 这个期间的样式会受到animation-fill-mode取值的影响。

        • ⏳详述

          1. 当animation-fill-mode: none;时，

            • 表示等待期时，元素仍是初始状态时的样式。

          2. 当animation-fill-mode: backwards | both;时，

            • 表示等待期时，元素的样式为第一帧的样式。

              第一帧可能是@keyframes中的0%，也有可能是100%。

              ⏳详述

              1. 当animation-direction: normal | alternate;时，
                • 第一帧就是 0% 中定义样式。
              2. 当animation-direction: reverse | alternate-reverse;时，
                • 第一帧就是 100% 中定义样式。

          3. 当animation-fill-mode: forwards;时，

            • 元素保持初始样式。

      3. 动画执行期

        • 指的是等待期结束后，瞬间开始执行动画且一直执行到最后一帧。

      4. 完成期

        • 动画执行完最后一帧时，元素处于的状态(❗注意: \text{: }: 无限循环的动画没有完成状态)。

          • 这个期间的样式会受到{animation-fill-mode}取值的影响。

        • ⏳详述

          1. 当animation-fill-mode: none;时，

            • 表示动画结束，元素回归初始状态(瞬间回归，无动画效果)。

          2. 当 animation-fill-mode: backwards; 时，

            • 元素跳转至初始样式。

          3. 当animation-fill-mode: forwards | both;时，

            • 表示动画执行完最后一帧时，保持在最后一帧样式。

              最后一帧可能是@keyframes中的0%，也有可能是100%。

              ⏳详述

              1. 当animation-direction: normal;时，
                • 最后一帧总为100%样式。
              2. 当animation-direction: reverse; 时，
                • 最后一帧总为0%样式。
              3. 当animation-direction: alternate;时，
                • 且{animation-iteration-count}为
                  1. 奇数次{1、3、5…}时，最后一帧为100%，
                  2. 偶数次{2、4、6…}时，最后一帧为0%。
              4. 当animation-direction: alternate-reverse;时，
                • 且{animation-iteration-count}为
                  1. 奇数次{1、3、5…}时，最后一帧为0%，
                  2. 偶数次{2、4、6…}时，最后一帧为100%。

    • 得出结论

      1. animation-fill-mode: none;
        • 表示等待期和完成期时，元素样式都为初始样式。
      2. animation-fill-mode: backwards;
        1. 表示等待期时，元素保持{第一帧样式}，
        2. 而完成期时，元素跳转至{初始样式}。
      3. animation-fill-mode: forwards;
        1. 表示等待期时，元素保持{初始样式}，
        2. 而完成期时，元素保持{最后一帧样式}。
      4. animation-fill-mode: both;
        1. 表示等待期时，元素保持{第一帧样式}，
        2. 而完成期时，元素保持{最后一帧样式}。

Css3 animation-play-state 属性

用于规定动画是否正在运行或已暂停。

  • 🎯特性

    默认值running
    继承no
    JavaScript 语法object object.style.animationPlayState=“paused”

  • 语法

    • 🍬解析
      1. paused
        • 使得动画暂停。
        • 🔔提示
          • animation-play-state: paused;
          • 一般应用于
            1. 当鼠标经过时，动画停止，
            2. 鼠标移开时，动画继续执行。
      2. running
        • 使得动画正在运行。

Css3 animation 属性

一个简写属性，

  • 用于设置六个动画属性 ➾ 

    1. animation-name，
    2. animation-duration，
    3. animation-timing-function，

  • 🎯特性

    • 🍬解析

      animation-delay用于规定动画效果何时开始{以秒或毫秒计}。
      animation-iteration-count用于规定动画的播放次数。
      animation-direction用于规定动画的播放方向。
      animation-fill-mode用于规定
      ①当动画不播放时，②当动画延迟时，③当动画播放完成时，
      要应用到元素的样式。

间歇性无缝轮播图与无缝轮播图很相似，

  • 只是在无缝轮播图的基础上增加了停留时间，让它每隔多少秒之后再执行向上滚动的操作。

Css样式中的百分比都是相对于谁的？

  1. 正常定位的width、height。

    • width和height的百分比
      • 是分别根据{父级元素块的宽度和高度}来计算的。

  2. 相对定位的top、left。

    • 💫示例

      • top和left的百分比
        • 是分别根据{父级元素块的宽度和高度}来计算的。

  3. 绝对定位的width、height和top、left。

    • 💫示例

      • top、height和left、width的百分比
        • 是分别根据包含它的第一个不是static定位的元素的{高度和宽度}来计算的。
        • 🔔提示
          • static定位的百分比是根据{浏览器视口的高度和宽度}来计算的。

  4. 固定定位的top、left。

    • 💫示例

      • top和left的百分比
        • 是分别根据{浏览器视口的高度和宽度}来计算的。

  5. translate。

    • 💫示例

      • translate是根据{自身的宽高}来计算的。

  6. margin-top、margin-left、padding-top、padding-left。

    • 💫示例

      • margin-top、margin-left、padding-top、padding-left是根据{父级元素块的宽度}来计算的。

  7. border-radius。

    • 💫示例

      • border-radius是根据{自身的宽高}来计算的。

  • background-size。

    • 💫示例

      • background-size是根据{背景区的宽高}来计算的。

Css3 all 属性

用于将所有应用到此元素的属性重置为其默认值或继承值，除了unicode-bidi和direction属性。

  • 🎯特性

    默认值none
    继承无
    JavaScript 语法object.style.all=“initial”

  • 语法

    • 🍬解析
      1. none
        • 不对所有应用到此元素的属性进行重置。
      2. initial
        • 将所有应用到此元素的属性设置为其默认值。
      3. inherit
        • 将所有应用到此元素的属性设置为其父元素的属性值(继承值)。
      4. unset
        • 将所有应用到此元素的属性设置为其默认值或父元素的属性值(继承值)。
        • ⏳详述
          1. 如果是没有继承特性的Css属性 ➾ display，则使用默认值，
          2. 如果是具有继承特性的Css属性 ➾ visibility，则使用继承值。

  • 🔥案例

    1. 根属性
    1. all: none;

    1. all: initial;

    1. all: inherit;

    1. all: unset;

简介

animate.css是一个来自国外的Css3动画库，

  • 它预设了抖动{shake}、闪烁{flash}、弹跳{bounce}等多达60多种动画效果，
  • 虽然借助animate.css能够很方便、快速的制作Css3动画效果，
    • 但还是建议看看animate.css的代码，也许你能从中学到一些东西。

使用方法

    1. 引入本地文件

    • 🔉说明

      • 第一个animated是必须添加的样式名，第二个是指定的动画样式名。
      • 🔔提示
