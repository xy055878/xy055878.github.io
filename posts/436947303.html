001.云计算快捷键

糖丸2024-03-032024-03-25

云计算快捷键

一、Linux编辑器(Xshell)快捷键

⑴光标移动类快捷键

  1. 光标移动到行首: \text{: }: Ctrl + A
  2. 光标移动到行尾: \text{: }: Ctrl + E

⑵剪切(删除)类快捷键

  1. 把光标所在位置到行首的内容剪切(删除){👈}: \text{: }: Ctrl + U

    • 🔥案例

  2. 把光标所在位置到行尾的内容剪切(删除){👉}: \text{: }: Ctrl + K

    • 🔥案例

⑶粘贴类快捷键

  • 粘贴上一次剪切(删除)的内容: \text{: }: Ctrl + Y

⑷功能性快捷键

  1. 清屏: \text{: }: Ctrl + L
  2. 补全「已存在的」命令、路径名: \text{: }: Tab
    • 🔔提示
      1. 按一下Tab可以补全。
      2. 按两下Tab可以显示当前路径下的内容(譬如: \text{: }: 目录、文件等)。
  3. 终止当前行输入的命令: \text{: }: Ctrl + C

⑸❗巨坑类快捷键

  1. 锁屏: \text{: }: Ctrl + S

    • 🔥案例

  2. 解锁: \text{: }: Ctrl + Q

    • 🔥案例

  3. 让当前正在运行的命令，后台挂起: \text{: }: Ctrl + Z

⑹查找历史命令类快捷键

  1. 上一个命令: \text{: }: ↑
  2. 下一个命令: \text{: }: ↓

⑺Xshell特有快捷键

  1. 复制: \text{: }: Ctrl + insert
  2. 粘贴: \text{: }: Shift + insert
  3. 登出系统: \text{: }: Ctrl + D
  4. 新建会话: \text{: }: Alt + N

⑻Vmware快捷键

  • 登出系统: \text{: }: Ctrl + D

二、vi/vim编辑器的基本使用

⑴vi/vim编辑器的工作模式

通过vi/vim命令打开文件后进入的模式是命令模式。

1.命令模式

按Esc键进入。

2.编辑模式

按i/a/o键进入。

  1. vi ➾ -- INSERT --
  2. vim ➾ -- 插入 --

3.末行模式

在命令模式下，按:键进入 ➾ 光标位于末行。

三、vim编辑器的快捷键

⑴命令模式下使用

1.光标移动类快捷键

  1. 行间调整

    1. 上/下行调整: \text{: }: ↑ / ↓

    2. 移动到最后一行开头: \text{: }: G 或 Shift + g

    3. 移动到第1行开头: \text{: }: gg

  2. 列间调整

    1. 左/右列调整: \text{: }: ← / →

    2. 移动到某一行开头: \text{: }: ngg

      • 🔔提示

        • n∈正整数。

      • 🔥案例

        • 移动到第100行开头: \text{: }: 100gg

    3. 光标移动到行首: \text{: }: ^(数字6)

    4. 光标移动到行尾: \text{: }: $(数字4)

2.复制、剪切(删除)、粘贴类快捷键

  1. 复制当前行: \text{: }: yy

  2. 剪切(删除)当前行: \text{: }: dd

  3. 粘贴到当前行后: \text{: }: p

3.其他类快捷键

  • 撤销: \text{: }: u

⑵末行模式下使用

  1. 退出不保存文件: \text{: }: :q

    • ❗注意

      • 输入内容再删除将 ➾ 无法成功。

      • 🔥案例

  2. 强制退出不保存文件: \text{: }: :q!

    • ❗注意

      • 输入内容再删除则 ➾ 可以成功。

      • 🔥案例

  3. 保存文件文件: \text{: }: :w

  4. 保存并退出文件: \text{: }: :wq

  5. 显示行号/ \text{/ }/ :set nu 或 :set set number

  6. 取消显示行号/ \text{/ }/ :set nonu

  7. 向「下」搜索你要的内容: \text{: }: /

    1. 继续向「下」搜索: \text{: }: n

    2. 继续向「上」搜索: \text{: }: N

  8. 向「上」搜索你要的内容: \text{: }: ？

    1. 继续向「下」搜索: \text{: }: n

    2. 继续向「上」搜索: \text{: }: N

  9. 临时取消语法高亮/ \text{/ }/ :noh
