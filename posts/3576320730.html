CSS 动画

    • 一些 CSS 属性是可以有动画效果的，

      • 这意味着它们可以用于动画和过渡。

    • 动画属性可以逐渐地

      • 从一个值变化到另一个值，
        • 🍒例如
          • 尺寸大小、数量、百分比和颜色。
