样式绑定

参数

一些指令能够接收一个“参数”，而参数在指令名称之后以冒号{:}表示。

  • 🍒例如
    1. v-bind:href="url"
      • 属性{href}就是参数。
      • ✨作用
        • 告知 v-bind 指令，将该元素的 href 属性与表达式 url 的值绑定。
    2. v-on:click="change"
      • 监听的事件名{click}就是参数。
      • ✨作用
        • 告知 v-on 指令，为该元素绑定点击事件change。

v-bind指令

用于将 Vue 实例中初始化数据{data}中的属性名，绑定到 HTML 元素的属性上 ➾ 动态绑定属性

  • class 和 style 是 HTML 元素的属性，用于设置元素的样式，我们可以用 v-bind 来设置样式属性。

  • v-bind 在处理 class 和 style 时，

    • 专门增强了它，表达式的结果类型可以是{字符串、对象、数组}。
