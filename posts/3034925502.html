2.PS1的常用参数及含义
  1. \d
    • 代表日期，格式为weekday month date，🍒例如: \text{: }: “Mon Aug 1”。
  2. \H
    • 代表完整的主机名称。
  3. \h
    • 代表仅取主机名中的第一个名字。
  4. \t
    • 代表显示时间为24小时格式，🍒例如: \text{: }: :HH:MM:SS
  5. \T
    • 代表显示时间为12小时格式。
  6. \A
    • 代表显示时间为24小时格式，🍒例如: \text{: }: :HH:MM
  7. \u
    • 代表当前用户的账号名称。
  8. \v
    • 代表Bash的版本信息。
  9. \w
    • 代表完整的工作目录名称。
  10. \W
    • 代表利用basename取得工作目录名称，只显示最后一个目录名。
  11. #
    • 代表下达的第几个命令。
  12. $
    • 代表提示字符，如果是root用户，提示符为 # ，普通用户则为$

二、Bash的特性

⑴按Tab键补全

当使用Windows查找一个目录层级特别多的文件时，打开的效率会非常的慢。

  • 但如果使用Linux查找一个目录层级特别多的文件时，可以通过Tab键快速补全。
  • 🔔提示
    1. 按一次Tab键可以补全 ➾ 内容(命令、选项、参数{譬如: 路径})。
    2. 按两次Tab键可以列出 ➾ 当前路径下所有以「目标」开头的内容。
      • ⏳详述
        • 如果按一次Tab键没有反应，
        • 那么需要按两次Tab键，列出 ➾ 当前路径下所有以「目标」开头的内容。
  • 🪄总结
    • 按一次可补全内容，按两次可显示具有相同点的内容。

      • 包括 ➾ 当前选中的内容。

3.粘贴类快捷键

  • 粘贴上一次剪切(删除)的内容: \text{: }: Ctrl + y

4.功能性快捷键

  1. 注释: \text{: }: #

    1. 加上#的命令，不会执行。但是在历史记录中 ➾ 存在。

    2. 不像Ctrl + c终止掉的命令在历史记录中 ➾ 不存在。

    3. 🔥案例

  2. 补全内容: \text{: }: Tab

  3. 清屏: \text{: }: Ctrl + l

  4. 退出登录(注销用户): \text{: }: Ctrl + d

  5. 将当前正在运行的命令 ➾ 终止: \text{: }: Ctrl + c

  6. 将当前正在运行的命令 ➾ 暂停，挂至后台: \text{: }: Ctrl + z

  7. 将上一条命令最后一个空格「后面的内容」，补全至当前命令行: \text{: }: Esc + .
