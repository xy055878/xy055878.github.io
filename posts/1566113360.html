事件修饰符

Vue 为 v-on指令提供了事件修饰符来处理 DOM 事件细节，代替了 ➾ 

  • 譬如: \text{: }: event.stopPropagation() ➾ 阻止事件冒泡等方法。

  • 🔔提示

    • 修饰符由以点{.}开头的指令后缀表示。

    • ⏳详述

      1. .stop ➾ 阻止事件冒泡。

        • 🔥案例

          • 阻止单击事件冒泡。

      2. .prevent ➾ 阻止浏览器默认行为。

        • 🔥案例

          • 提交事件不再重载页面。

      3. .capture ➾ 阻止事件捕获。

      4. .self ➾ 只监听触发该元素本身(而不是子元素)的事件。

      5. .once ➾ 每次页面重载后，只会触发一次事件。

        • 🔥案例

          • 单击事件只能点击一次。

    • 其他🔥案例

      1. 只有修饰符 ➾ 不绑定事件。

      2. 修饰符串联。

        • ❗注意

          • 使用事件修饰符时，顺序很重要 ➾ 不同的代码顺序将会产生不同的效果。
            1. 用 v-on:click.prevent.self 会阻止所有的点击事件。
            2. 用 v-on:click.self.prevent 只会阻止对元素自身的点击事件。

      3. 模态框

        1. 方案一 ➾ <div id="center" @click.stop>

        2. 方案二 ➾ @click.self="isShow=false"

按键修饰符

在监听键盘事件时，我们经常需要检查详细的按键。

  • 而Vue 允许为 v-on指令在监听键盘事件{v-on:keyup}时，添加按键修饰符。

键盘修饰符

  1. .enter ➾ 回车键。

    • 🔥案例

  2. .tab ➾ 制表键。

  3. .delete ➾ 删除键和退格键。

  4. .esc ➾ 取消键。

  5. .space ➾ 空格键。

  6. .up ➾ 上键。

  7. .down ➾ 下键。

  8. .left ➾ 左键。

  9. .right ➾ 右键。

按键码

使用按键对应的ASCII字符集触发按键事件 ➾ 键盘按键与ASCII字符集对照表。

  • 🔥案例

    • 🔔提示
      • 13 ➾ enter{回车键}。

系统修饰键

  1. .ctrl

    • 🔥案例

      • Ctrl + Click。

  2. .alt

  3. .shift

  4. .meta

    • 🔔提示
      1. 在 Mac 系统键盘上，meta 对应 command 键{⌘}。
      2. 在 Windows 系统键盘，meta 对应 Windows 徽标键{⊞}。

  5. .exact

    • .exact允许你控制由精确的{系统修饰符组合}触发的事件。

    • 🔥案例

      1. Alt或Shift键被按下时，也会触发事件。

      2. 有且只有Ctr键被按下的时，才会触发事件。

      3. 没有任何{系统修饰符}被按下的时，才会触发事件。

鼠标修饰符

  1. .left ➾ 鼠标左键事件。

  2. .right ➾ 鼠标右键事件。

  3. .middle ➾ 鼠标中间滚轮事件。
