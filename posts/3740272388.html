Css 网格布局

Grid布局即网格布局，是一种新的Css布局模型，由一个父元素及一个或多个子元素组成

  • 🎯特性

    • 擅长将一个页面划分为几个主要区域，
      • 以及定义这些区域的{大小、位置、层次}等关系。
    • 号称是最强大的的Css布局方案，是目前唯一的一种Css二维布局。

  • 💫图例

    • 如下是一个简单的网页布局，它使用了网格布局，包含六列和三行➾

Grid布局与Flex布局的区别

  • Flex布局是一维布局，

    • 只能处理{一个维度}上的布局，🪶即一行或一列。

  • Grid布局是二维布局，

    • 能处理{二个维度}上的布局，
      • 其将容器划分成了行和列，进而产生了一个个的网格，
      • 使得网格元素可以放在行和列的相关位置上，从而达到了布局的目的。

相关概念介绍

1)网格容器

将display属性的属性值设置为grid或inline-grid的HTML元素，称为网格容器。

  • 🔔提示

    • grid容器和inline-grid容器的区别在于，

      1. grid容器{块网格}保持块级元素特性，

        • 不和行内元素一行显示。『容器内部从上向下排列』

Css grid-template-rows 属性

用于设置网格布局中的行数及高度。

  • 该属性值是一个以{空格}分隔的列表，

    • 其中
      1. grid-template-rows属性值的{数量}表示 ➾ 网格的行数，
      2. grid-template-rows属性的{每个值}表示 ➾ 对应行的高度。

  • 🎯特性

    默认值none
    继承no
    JavaScript 语法object.style.gridTemplateRows=“50px 200px”

  • 语法

    • 🍬解析

      1. none

        • 不指定行的高度。

      2. auto

        • 网格行的高度由网格容器的高度和网格行中网格元素内容的高度决定，
        • 🪶即自动获取剩余高度。
          • 🍒例如
            • grid-template-rows: 100px auto 100px;
              • 第二行高度为剩余高度。

      3. max-content

        • 每行的高度设置为该行中最大网格元素的高度。

      4. min-content

        • 每行的高度设置为该行中最小网格元素的高度。

      5. length

        • 可选值 ➾ 

          1. 绝对大小，根据行数确定值的个数。

            • 🔥案例

            • 🔔提示

              • 0 ➾ 默认值。

          2. 百分比，根据行数确定值的个数。

            • 🔥案例

            • 🔔提示

              • % ➾ 相对于网格容器的高度。

      6. initial

        • 将此属性设置为默认值，阅读关于initial。

      7. inherit

        • 从父元素继承该属性，阅读关于inherit。

grid-template-rows丨columns的相关关键字
1)fr 关键字

一个fr单位代表网格容器中，剩余空间的一等份。

  • 🔔提示

    • 可以理解为份数，🪶即把剩余空间分为多少份。

  • 🍒例如

    • grid-template-columns: 150px 1fr 2fr;
      • 👁️‍🗨️分析
        1. 第1列的宽度为150px，
        2. 剩余空间按{1:2}的比例，分给另外2列，
          • 第2列一份，第3列两份，共三份。
        3. 当无剩余长度时，则不占空间🪶即为 ➾ 0。
2)auto 关键字
  1. 设置网格元素的行高由网格容器的高度和网格元素内容的高度决定，🪶即为剩余高度，
  2. 设置网格元素的列宽由网格容器的宽度和网格元素内容的宽度决定，🪶即为剩余宽度。
3)% 关键字
  1. 设置网格元素的行高{相对于}网格容器的高度，
  2. 设置网格元素的列宽{相对于}网格容器的宽度。

  • 🔥案例

    • 👁️‍🗨️分析

      • 上面这段代添加了六个列。
        1. 第一列的宽度为剩余宽度，
        2. 第二列的宽度50px，
        3. 第三列的宽度是它容器的10%，
        4. 最后三列，
          • 将剩余宽度平均分成四份，
            1. 第四列占两份，
            2. 第五列占一份，
            3. 第六列占一份。

    • 🎭图示

4)auto-fill关键字

自动填充，🪶即将网格容器的空间均匀划分给网格元素，

  • 当网格容器的大小{大于}各个网格元素之和时，

    • 剩余空间以空白显示。

  • ❗注意

    • 如果容器{无法使}所有网格元素放在同一行，那么余下的网格元素{将移至}新的一行。
5)auto-fit关键字

自动适应，🪶即将网格容器的空间均匀划分给网格元素，

  • 当网格容器的大小{大于}各个网格元素之和时，

    • 将各个网格元素均匀拉伸至合适的大小，导致 ➾ 无剩余空间。

  • ❗注意

    • 如果容器{无法使}所有网格元素放在同一行，那么余下的网格元素{将移至}新的一行。

  • 🔥案例

    • auto-fit关键字和auto-fill关键字的区别。

Css grid-template-areas 属性

grid-template-areas 属性用于设置网格布局，grid-area 属性可以对当前网格元素进行命名。

  • 🔔提示

    1. 命名的网格元素可以通过网格容器的 grid-template-areas 属性来引用，
    2. 每行由单引号内的grid-area属性值定义，以{空格}分隔，
    3. 点号{.}表示没有名称的网格元素。

    1. 区域模板只能是矩形，而不能是不规则图形，

    默认值none
    继承no
    JavaScript 语法object.style.gridTemplateAreas=“. . . myArea myArea”

    • 🍬解析
      1. none
        • 不引用已经命名的网格元素。
      2. itemnames
        • 每行以引号{''}分割，每行内以{空格}分割的自定义网格元素名列表，
          • 🪶即指定每一网格行和网格列的显示方式。

  • 🔥案例

    1. item1元素命名为“myArea”, 并跨越五个网格列。

      • 🔔提示
        • 网格元素的命名不用加引号{''}。
        • 🍒例如
          • grid-area: myArea;

    2. 设置item1元素跨越 1 行 2 列。

      • 🔔提示

        1. 每行由单引号内的grid-area属性值{myArea}定义，以空格分隔，
        2. 点号{.}表示没有名称的网格元素。
          • 如果某些区域不需要使用，则可以使用点号表示。
          • ❗注意
            • 点号必须相邻的。

        • 🥦补充

          • grid-template-areas: 'myArea myArea myArea';

      • 🔔提示

    4. 命名所有网格元素，并制作一个网页模板。

        • 上面中任意一个grid-areas属性值都可以替换为点号{.}，表示该区域不使用。

                    • auto关键字对应的宽度是大于100px的，不妨用233px代替，

                    • 因此，此时fit-content(100px)的尺寸就是设置的100px。

                    • 套用公式

                  • 随着容器宽度变小，

                    • auto关键字对应的宽度也越来越小，

                      • 可以一直小到min-content宽度，
                        • 这里min-content的宽度值应该是100px对应的5个字符的宽度。
                        • 我量了一下，在我的电脑上是38px。

                    • 套用公式

      3. max-content

        • 每行的高度设置为该行中最大网格元素的高度。

      4. min-content

        • 每行的高度设置为该行中最小网格元素的高度。

      5. minmax(min, max)

        • 设置网格行的默认大小局限于 ➾ 大于等于{min}且小于等于{max}。

      6. length

        • 使用自定义的值设置网格行的默认大小，默认值 ➾ 0。

  • 🔥案例

      2. fit-content(limit)

        • 一言以蔽之，“尺寸由内容决定，内容越多尺寸越大，但不超过限定的尺寸”。

        • 相当于公式min(min(auto, limit), min(auto, max-content))

          • 🍬解析
            1. min()
              • JS中Math.min()的意思，🪶即取小值。
            2. auto
              • {auto}关键字对应的尺寸。
            3. max-content
              • {max-content}对应的尺寸。

        • 语法

          • 🍬解析
            1. length
              • 长度值。
            2. percentage
              • 百分比值。

      3. max-content

        • 每列的宽度设置为该列中最大网格元素的宽度。

      4. min-content

        • 每列的宽度设置为该列中最小网格元素的宽度。

      5. minmax(min, max)

      6. length

        • 如果剩余空间为负或只有一个网格元素，则该值等同于center。

        • 否则，

          1. 均匀排列每个网格元素，

        • 均匀排列每个网格元素且每个网格元素之间的间隔相等。
