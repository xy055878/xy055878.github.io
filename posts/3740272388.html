Css 网格布局🏅①

Grid布局即网格布局，是一种新的Css布局模型，由一个父元素及一个或多个子元素组成

  • 🎯特性

    • 擅长将一个页面划分为几个主要区域，
      • 以及定义这些区域的{大小、位置、层次}等关系。
    • 号称是最强大的的Css布局方案，是目前唯一的一种Css二维布局。

  • 💫图例

    • 如下是一个简单的网页布局，它使用了网格布局，包含六列和三行➾

Grid布局与Flex布局的区别

  • Flex布局是一维布局，

    • 只能处理{一个维度}上的布局，🪶即一行或一列。

  • Grid布局是二维布局，

    • 能处理{二个维度}上的布局，
      • 其将容器划分成了行和列，进而产生了一个个的网格，
      • 使得网格元素可以放在行和列的相关位置上，从而达到了布局的目的。

相关概念介绍

1)网格容器

将display属性的属性值设置为grid或inline-grid的HTML元素，称为网格容器。

  • 💫示例

  • 🔔提示

    • grid容器和inline-grid容器的区别在于，

      1. grid容器{块网格}保持块级元素特性，

        • 不和行内元素一行显示。『容器内部从上向下排列』

        • 🔥案例

      2. inline-grid容器{行内网格块}保持行内块元素特性，

        • 因此可以和图片文字一行显示。『容器内部从左向右排列』

      1. 一般给<div>这类块级元素元素设置display: grid;

      2. 一般给<span>这类行内元素设置display: inline-grid;

        • 🔔提示

          • 对行内元素设置display: inline-grid;

            • 可将行内元素转化为块级元素

          • 🔥案例

            1. <span> + display: grid丨inline-grid;
            1. <span> - display: grid丨inline-grid;;

5)网格线

网格容器内部，划分网格的线，称为网格线。

  • 🔔提示

    • 水平网格线划分出{网格行}，垂直网格线划分出{网格列}。

  • 正常情况下，

    1. n行有n+1根水平网格线，
    2. m列有m+1根垂直网格线，

  • 🍒例如

    • 一个 2行3列{2×3} 的网格，

      • 共有3根水平网格线{黑色圆圈标记}和4根垂直网格线{灰色圆圈标记}。
      • 🔔提示
        • 网格布局会为我们创建编号的网格线，来让我们来定位每一个网格元素。
        • 网格线的编号顺序取决于文章的书写模式。
          • ⏳详述
            1. 在{从左至右}书写的语言中，
              • 编号为①的网格线位于最左边。
            2. 在{从右至左}书写的语言中，
              • 编号为①的网格线位于最右边。

6)网格单元

网格容器内部，网格行和网格列的交叉区域，称为网格单元，

  • 在一个网格元素中，最小的单位就是一个网格单元，

    • 🔔提示
      • 从概念上来讲，其实它同表格的一个单元格很像。

  • 正常情况下，n行m列 ➾ 会产生n×m个网格单元。

    • 🍒例如

      • 2行3列 ➾ 会产生6个网格单元。

网格容器属性

Css grid-template-rows 属性🏅③

用于设置网格布局中的行数及高度。

  • 该属性值是一个以{空格}分隔的列表，

    • 其中
      1. grid-template-rows属性值的{数量}表示 ➾ 网格的行数，
      2. grid-template-rows属性的{每个值}表示 ➾ 对应行的高度。

  • 🎯特性

    默认值none
    继承no
    JavaScript 语法object.style.gridTemplateRows=“50px 200px”

  • 语法

    • 🍬解析

      1. none

        • 不指定行的高度。

      2. auto

        • 行的高度由容器的高度和行中网格元素内容的高度决定，
        • 🪶即自动获取剩余高度。
          • 🍒例如
            • grid-template-rows: 100px auto 100px;
              • 第二行高度为剩余高度。

      3. max-content

        • 每行的高度设置为该行中最大网格元素的高度。

      4. min-content

        • 每行的高度设置为该行中最小网格元素的高度。

      5. length

        • 可选值 ➾ 

          1. 绝对大小，根据行数确定值的个数。

            • 🔥案例

            • 🔔提示

              • 0 ➾ 默认值。

          2. 百分比，根据行数确定值的个数。

            • 🔥案例

            • 🔔提示

              • % ➾ 相对于网格容器的高度。

      6. initial

        • 将此属性设置为默认值，阅读关于initial。

      7. inherit

        • 从父元素继承该属性，阅读关于inherit。

grid-template-rows丨columns的相关关键字🏅④
1)fr 关键字

设置网格行高(列宽)占据剩余空间的一个比例。

  • 如果两列的宽度分别为1fr和2fr，就表示后者是前者的两倍。
  • 🍒例如
    • grid-template-columns: 150px 1fr 2fr;
      • 表示第一列的宽度为150px，
      • 剩余的长度按1:2分给另外2列，无剩余长度时则为 ➾ 0。
2)auto 关键字

设置网格行高(列宽)由容器的高度(宽度)和列中网格元素内容的高度(宽度)决定，🪶即为剩余高度(宽度)。

3)% 关键字

设置网格行高(列宽)相对于网格容器的高度(宽度)。

  • 🔥案例

    • 👁️‍🗨️分析

        4. 最后三列，
          • 将剩余宽度平均分成四份，
            1. 第四列占两份，
            2. 第五列占一份，
            3. 第六列占一份。

Css grid-template-areas 属性🏅⑥

grid-template-areas 属性用于设置网格布局，grid-area 属性可以对网格元素进行命名。

  • 🔔提示

    1. 命名的网格元素可以通过容器的 grid-template-areas 属性来引用，
    2. 每行由单引号内的grid-area属性值定义，以空格分隔，
    3. 点号{.}表示没有名称的网格项。

  • ❗注意

    1. 区域模板只能是矩形，而不能是不规则图形，
    2. 声明中的每一行都必须具有相同数量的单元格。

  • 🎯特性

    默认值auto
    继承no
    JavaScript 语法object.style.gridTemplateAreas=“. . . myArea myArea”

  • 语法

    • 🍬解析
      1. none
        • 不引用已经命名的网格。
      2. itemnames
        • 每行以引号{''}分割，每行内以空格的网格名列表，🪶即指定每列和行的显示方式。

  • 🔥案例

    1. item1元素命名为“myArea”, 并跨越五列。

      • 🔔提示
        • 区域的命名不用加引号{''}。

    2. 设置item1元素跨越 1 行 2 列。

      • 🔔提示

        1. 每行由单引号内的grid-area属性值{myArea}定义，以空格分隔，
        2. 点号{.}表示没有名称的网格元素。
          • 如果某些区域不需要使用，则可以使用点号表示。
          • ❗注意
            • 点号必须相邻的。

        • 🥦补充

          • grid-template-areas: 'myArea myArea myArea';

    3. 设置item1元素跨越 2 行 2 列。

      • 🔔提示

    4. 命名所有网格元素，并制作一个网页模板。

Css grid-area 属性🏅⑥

grid-area属性是如下属性的简写属性 ➾ 

  1. grid-row-start和grid-column-start属性，
  2. grid-row-end和grid-column-end属性。
    • 🔔提示
      1. grid-area 属性也可以对当前网格元素进行命名。
      2. 命名的网格元素可以通过容器的 grid-template-areas 属性来引用。

🎯特性

默认值auto / auto / auto / auto
继承no
JavaScript 语法object.style.gridArea=“1 / 2 / span 2 / span 3”

语法

  • 🍬解析
    1. grid-row-start
      • 指定在哪一行开始显示当前网格元素。
    2. grid-column-start
      • 指定在哪一列开始当前显示网格元素。
    3. grid-row-end
      • 指定哪一行停止显示当前网格元素或跨越多少行。
    4. grid-column-end
      • 指定哪一列停止显示当前网格元素或跨越多少列。
    5. itemname
      • 网格名，🪶即对当前网格元素进行命名。
